COSC 301 Project:

These files include the data cleaning, and analysis questions we aimed to answer during our investigation.

To run, you will need the games.json file, which can be gathered by running code found here https://github.com/FronkonGames/Steam-Games-Scraper. This dataset includes everygame found on Steam, with useful statistics like player count, price, and rating. The Steam game scraping code is liscenced under MIT and is free to use.

After running the data_cleaning notebook, the date is stored inside of SQL tables locally (sqlite3) and then you can run the analysis notebook which will export the findings as CSVs.
